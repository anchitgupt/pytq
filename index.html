How It Works

Data-oblivious quantization

No calibration data. No fine-tuning. Just math. TurboQuant exploits the geometry of the unit sphere to achieve provably near-optimal compression.

01
Normalize
Store the L2 norm. Map the vector onto the unit sphere Sd-1.
02
Rotate
Apply a random orthogonal matrix. Each coordinate now follows a known Beta distribution.
03
Quantize
Scalar quantize each coordinate to the nearest Lloyd-Max centroid. Store as b-bit indices.
04
Dequantize
Look up centroids, inverse rotate, rescale by stored norm. Done.
Dmse ≤ (√3 · π / 2) · 4−b  —  within 2.72x of the information-theoretic lower bound

Get Started

Three lines to compress

$ pip install pytq click to copy
# Quantize any vector in 3 lines from pytq import TurboQuantMSE quantizer = TurboQuantMSE(dim=128, bits=2) q = quantizer.quantize(x) # compress x_hat = quantizer.dequantize(q) # decompress

# Drop-in KV cache for HuggingFace models from pytq import TurboQuantKVCache cache = TurboQuantKVCache(bits=2.5, head_dim=128) # Use as past_key_values in your generation loop model.generate(inputs, past_key_values=cache)

# Unbiased inner products with TurboQuant_prod from pytq import TurboQuantProd quantizer = TurboQuantProd(dim=128, bits=3) q = quantizer.quantize(x) x_hat = quantizer.dequantize(q) # E[<y, x_hat>] = <y, x> -- mathematically guaranteed

# Run benchmarks $ python -m benchmarks.run_all --quick $ python -m benchmarks.run_all --with-quality \ --quality-model meta-llama/Llama-3.2-1B
